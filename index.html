  • 📌 사용 기술
    • react
    • redux
    • local-storage
    • axios
    • json-server

  • 📌 배운점
    • react: 반복되는 코드를 컴포넌트화, jsx문법 사용 또한, 리액트 훅 usestate, useeffect역할 사용방법 학습

    • json-server: 가짜 API서버를 구축해 카테고리별 상품 데이터 저장

    • - 항생제, 항암제 등 부작용이 높은 약물은 수액 주입 속도가 중요

    • arduino

    • android studio
    • java

    • 환자 정보 등록
    • 수액 투여 속도가 적정 속도에서 벗어나면 간호사에게 알림

    • 꽉찬 배표팩을 100%로 무게 센서를 이용해 받아온 값을 0 30 60 100 퍼센트로 표현 후 100 퍼센트일 경우 '환자 확인 요망' 알림 뜨기

슈팅게임

❌

Java 기초 지식을 학습한 후 만든 간단한 슈팅 게임입니다.

  • 📌 게임 방법
    • 1. 방향키로 잠수함 위치 이동
    • 2. 스페이스 클릭 시, 물풍선 폭격
    • 3. 물고기와 잠수함이 닿으면 게임종료

    • 5. 물풍선으로 물고기를 한번 맞추면 물풍선 맞은 표시
    • 6. 물풍선으로 물고기를 두번 맞추면 score 증가 +50
