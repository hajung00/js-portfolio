  • 가짜 API서버를 구축하여 만든 쇼핑몰
  • 사용 기술
    • react
    • redux
    • local-storage
    • axios
    • json-server
  • 배운점
    • react: 반복되는 코드를 컴포넌트화, jsx문법 사용 또한, 리액트 훅 usestate, useeffect역할 사용방법을 학습
    • redux: 스토어를 통해 데이터 관리, 리듀서를 통한 데이터 변경 (적용 -> 바구니 추가, 수량변경)
    • local-storage: 로컬스토리지 메소드를 이용해 방문한 상품을 키-값으로 저장
    • axios: 동기 통신,비동기 통신,promise객체에 대해 학습 (적용 -> get메소드를 이용해 상품의 목록을 불러옴)
    • json-server: 가짜 API서버를 구축해 카테고리별 상품 데이터 저장
